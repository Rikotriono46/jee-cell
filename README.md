# 🏪 Jee Cell - Agen BRILink & Toko Online

Website toko serba ada dengan layanan BRILink, pulsa, voucher game, token listrik, alat tulis, jajanan, dan cetak foto.

## Fitur

- 🏦 **BRILink** - Transfer, tarik/setor tunai, bayar tagihan
- 📱 **Pulsa & Data** - Semua operator
- 🎮 **Voucher Game** - ML, FF, PUBG
- ⚡ **Token Listrik** - PLN prabayar
- ✏️ **Alat Tulis** - Buku, pulpen, pensil
- 🍦 **Jajanan & Minuman** - Snack, es, kopi
- 📸 **Foto & Print** - Cetak foto, print, fotokopi

## Admin Panel

- CRUD produk, kategori, pengumuman
- Pengaturan toko (nama, alam, jam buka, WhatsApp)
- Dashboard statistik

## Tech Stack

- **Backend:** Flask + SQLite
- **Frontend:** HTML/CSS/JS, Font Awesome
- **Server:** Gunicorn
